# FLOYDVANAVIE — GitHub Pages Deploy

## Files
- `index.html` — your site (edit text directly in this file)
- `favicon.ico` — browser tab icon

## Publish steps
1) Upload both files to your repo root (`lincolnm272-code/Floydvanavie-`).
2) Repo **Settings → Pages** → Source: `Deploy from a branch`.
   - Branch: `main`
   - Folder: `/ (root)`
3) Save and wait ~1 minute. Your site is at:
   https://lincolnm272-code.github.io/Floydvanavie-/

To edit the text, open `index.html` on GitHub, click **Edit**, make your changes, and **Commit**.
